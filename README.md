## Overview

This project displays a simple QR code card with:

- A QR code image  
- Heading: *Improve your front-end skills by building projects*  
- Description: *Scan the QR code to visit Frontend Mentor and take your coding skills to the next level*  

The design is fully responsive and uses Flexbox for **centered alignment**. Shadows, rounded corners, and spacing create a modern card appearance.

---

## Features

- Fully responsive design  
- Centered content using Flexbox  
- Modern card with rounded corners and shadow  
- Scalable QR code image  
- Semantic HTML structure  
- Easy to customize and extend
